# Student Records Program

#### A program which manages student records.
#### A record contains: 
1. name
2. age
3. courses
4. grades dict (a method decorated as property) 

<b>Lookups:</b>
1. students:
    - add and remove them
    - num of students
    - student list

2. grades:
    - per student
    - per course 

Note: Course object is unknowm to User.
